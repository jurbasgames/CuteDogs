## About
CuteDogs is and image app using TheDogAPI

## Requirements
    
Before start, check if you have:

- Yarn

## Starting


### Clone the folder.
```
$ git clone https://github.com/jurbasgames/CuteDogs
```
### Access it.
```
$ cd CuteDogs
```
### Install dependecies
```
$ yarn install
```

### Run

```
$ yarn start
```
